#  Network Scanner

##  Description

A Python-based network scanner that detects open ports on a target system.

##  Features

* Scans ports from 20 to 1024
* Identifies open ports
* Fast and simple scanning

##  Technologies Used

* Python
* Socket Programming

##  Usage

```bash
python main.py <target>
```

### Example

```bash
python main.py scanme.nmap.org
```

##  What I Learned

* How network ports work
* Basics of port scanning
* Socket programming in Python
